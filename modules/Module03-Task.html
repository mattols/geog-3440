Perform the following tasks.

  1. What species of iris has a sepal length greater than 7 cm and petal length less than or equal to 2 cm? List the name and upload the code used.

  3. Recreate the plot below, including colors, labels, and legend (trend line is optional and can be added using the abline function - see tip below plot).

Environmental Science

High levels can be particularly worrisome when levels exceed limits by more than 3 consecutive weeks.   2. Report the total number of occurrences (weeks) for each site in which the limits for nitrate and phosphate were exceeded. Show this information in one or more plots.
